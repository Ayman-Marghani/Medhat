# Medhat
### Commands to install all needed python packages
- open the app folder "Medhat" with the terminal then run the following bash commands
```bash
cd medhat_python
```
```bash
pip install -r requirements.txt
```
```bash
python -m spacy download en_core_web_md
```
### Run "Medhat.py"
```bash
python Medhat.py
```
